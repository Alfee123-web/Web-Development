# Web Development Journey 🚀

Welcome to my web development repository! This repository serves as a comprehensive archive of my learning journey, code practices, and full-stack projects built while mastering front-end and back-end technologies.

## 📂 Repository Structure

The repository is organized into structured modules based on topics and projects:

| Folder Name | Description |
| :--- | :--- |
| `📁 HTML` / `HTML2` / `HTML3` | Core concepts of semantic HTML5, structures, tables, and web forms. |
| `📁 CSS` to `CSS6` | UI Styling, Flexbox, Grid layouts, Media Queries for responsiveness, and advanced animations. |
| `📁 Bootstrap` | Utilizing the Bootstrap framework for fast, mobile-first responsive web design. |
